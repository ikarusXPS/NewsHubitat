## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding (`gap-1`) |
| sm | 8px | Compact element spacing (`gap-2`, `p-2`) |
| md | 16px | Default element spacing (`gap-4`, `p-4`) |
| lg | 24px | Section padding (`p-6`) |
| xl | 32px | Layout gaps (`gap-8`) |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Article preview card padding: 12px (`p-3`) for compact density within event panel
- LIVE badge inline padding: 6px (`px-1.5`) for compact indicator
- Timeline dot offset: 32px (`-left-8`) for timeline line alignment

| High | #ff6600 | Severity 6-7 |

### Accent Reserved For
- Primary CTA buttons (external link icon, "Artikel anzeigen" button)
- Event card hover states (border-color transition to #00f0ff)
- LIVE badge text (for non-critical recent events)
- Timeline connector glow on hover
- Globe/Map marker selection ring
- Panel section headers (icon color)

**Accent is NOT used for:** Body text, secondary buttons, disabled states, decorative elements.

## Component Contracts

### ArticlePreview Component (EVT-01, D-01, D-02, D-03)

**Purpose:** Inline article preview within EventDetailPanel.

**Visual Contract:**
- Container: `p-3 rounded-lg bg-gray-800/50 hover:bg-gray-800 transition-colors`
- Layout: Flexbox row, justify-between, items-center
- Title: 14px JetBrains Mono, #d1d5db (gray-300), truncate single line, hover: #00f0ff
- Metadata row: 10px JetBrains Mono, #6b7280 (gray-500), pipe-separated
- Metadata format: `{source.name} | {timeAgo} | {REGION}`
- External link icon: 16px, #6b7280, hover: #00f0ff, positioned right

**Interaction Contract:**
- Card click: No action (external link is separate)
- External link click: Opens `article.url` in new tab, `stopPropagation()` to prevent card selection

**Data Contract (from RESEARCH.md):**
```typescript
interface ArticlePreviewData {
  id: string;
  title: string;
  source: { name: string };
  publishedAt: string;  // ISO date
  perspective: PerspectiveRegion;
  url: string;
}
```

### LiveBadge Component (EVT-04, D-15)

**Purpose:** Indicate recent events (<30 min old).

**Visual Contract:**
- Container: `inline-flex items-center gap-1 px-2 py-0.5 rounded`
- Background: `rgba(255, 0, 68, 0.2)` (#ff0044/20)
- Border: 1px solid `rgba(255, 0, 68, 0.5)`
- Text: 9px JetBrains Mono, #ff0044, uppercase, font-bold, tracking-wider
- Pulsing dot: 6px circle, #ff0044 fill, `animate-pulse`

**Position Contract:**
- Timeline EventCard: Top-right corner of card, absolute positioned
- EventMap marker popup: Inline with timestamp
- Globe marker: Below title in tooltip

**Logic Contract:**
```typescript
const isRecent = Date.now() - new Date(timestamp).getTime() < 30 * 60 * 1000;
// Use useMemo to avoid recalculation on every render
```

### EventDetailPanel - Articles Section (EVT-01)

**Purpose:** Display related articles within existing EventDetailPanel.

**Visual Contract:**
- Section header: "Verwandte Artikel ({n})", 10px JetBrains Mono uppercase, #6b7280
- List container: `space-y-2`
- Each article: ArticlePreview component
- Max display: 10 articles initially, "Mehr anzeigen" button if >10

**Layout:**
```
+----------------------------------+
| VERWANDTE ARTIKEL (3)            |
+----------------------------------+
| [Article Preview 1]              |
| [Article Preview 2]              |
| [Article Preview 3]              |
+----------------------------------+
```

### HistoricalEventCard (EVT-02, D-05, D-06, D-07)

**Purpose:** Display historical events with bilingual support.

**Visual Contract:**
- Container: `glass-panel rounded-lg p-4 border border-[#00f0ff]/20 hover:border-[#00f0ff]/40`
- Date display: Large year badge, 20px JetBrains Mono bold, #00f0ff
- Title: 16px JetBrains Mono medium, white
- Description: 14px Inter, #9ca3af, line-clamp-3
- Category badge: Existing CATEGORY_CONFIG styling
- Location chip: MapPin icon + name, 10px JetBrains Mono, #00f0ff/70

**i18n Contract:**
```typescript
const language = useAppStore(state => state.language); // 'de' | 'en'
const title = event.title[language] || event.title.de; // Fallback to German
```

### GlobeView Event Integration (EVT-04, D-14)

**Purpose:** Display geo-events on 3D globe, synced with EventMap.

**Visual Contract:**
- Use existing marker styling from GlobeView.tsx
- LIVE badge: Not on globe markers (too small), only in tooltip
- Severity colors: Match CATEGORY_COLORS constant
- Selected marker: Increased size (1.5x), brighter glow

**Sync Contract:**
```typescript
// Same query as EventMap.tsx - ensures cache sharing
const { data: events } = useQuery({
  queryKey: ['geo-events'],
  queryFn: fetchGeoEvents,
  staleTime: 60_000,
  refetchInterval: 2 * 60_000,
});
```

## Animation Contracts

### LIVE Badge Pulse
```css
@keyframes pulse-glow {
  0%, 100% { opacity: 1; transform: scale(1); }
  50% { opacity: 0.5; transform: scale(0.8); }
}
/* Duration: 1.5s, ease-in-out, infinite */
```

### Event Panel Slide-In
```typescript
// Framer Motion
initial={{ opacity: 0, x: 300 }}
animate={{ opacity: 1, x: 0 }}
exit={{ opacity: 0, x: 300 }}
// Duration: spring with damping 25, stiffness 200
```

```css
}
/* Duration: 1.5s, ease-in-out, infinite */
```

## Accessibility Contract

### Keyboard Navigation
- Event cards: Tabbable, Enter/Space to select
- Article preview external link: Focusable, Enter to open
- Event panel close: Escape key, X button focusable
- Filter buttons: Tabbable, Space to toggle

### Screen Reader
- LIVE badge: `aria-label="Live event, updated within last 30 minutes"`
- Severity indicator: `aria-label="Severity: {n} out of 10"`
- External link: `aria-label="Open article in new tab"`
- Event panel: `role="dialog"`, `aria-labelledby` pointing to title

### Color Contrast
- All text meets WCAG AA on dark backgrounds
- Severity colors have sufficient contrast at opacity 1.0
- Links are distinguishable by underline on hover, not color alone
