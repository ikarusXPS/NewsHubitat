# Plan 41-03: Privacy.tsx vollständig + DE/EN/FR (DOC-02)

## Goal

Erweitere `Privacy.tsx` zur vollständigen Datenschutzerklärung. Aktuell stark gekürzt; Zielzustand spiegelt 1:1 die 24 Verarbeitungstätigkeiten + Drittanbieter-Tabelle + Retention-Schedule + Betroffenenrechte aus `docs/legal/PROCESSING-RECORDS.md`. Inhalte werden über i18n-Namespace `privacy` in DE/EN/FR ausgespielt.

## Steps

1. `apps/web/public/locales/de/privacy.json` (Master) mit vollständiger Sektion-Struktur erstellen:
   - `privacy.imprint.*` (Verantwortlicher, Anschrift, DSB-Kontakt — Werte aus 41-01)
   - `privacy.intro.*` (Geltungsbereich + Stand)
   - `privacy.activities.{key}.*` für jede der 24 Tätigkeiten (Bezeichnung + Zweck + Rechtsgrundlage + Kategorien + Empfänger + Drittlandübermittlung + Löschfrist)
   - `privacy.processors.*` (Drittanbieter-Tabelle als strukturierte Liste)
   - `privacy.retention.*` (Löschfristen-Übersicht)
   - `privacy.rights.*` (Art. 15-21 + entsprechende Endpoints)
   - `privacy.complaint.*` (Beschwerderecht Art. 77 — zuständige Aufsichtsbehörde Bundesland-abhängig)
   - `privacy.changes.*` (Änderungs-Hinweis)
2. `en/privacy.json` + `fr/privacy.json` via DeepL-Initial-Übersetzung; manuelles Review für rechtsrelevante Begriffe (Aufsichtsbehörde, Rechtsgrundlage, …)
3. `Privacy.tsx` rendert Struktur via `useTranslation('privacy')` mit map-loops über Tätigkeiten/Drittanbieter
4. Footer-Link prüfen: `/privacy` muss aus jedem Layout erreichbar sein (typisch im `Footer.tsx`)
5. ConsentBanner-Verlinkung prüfen: "Mehr erfahren" / "Datenschutz" linkt auf `/privacy`

## Verification

- Visual-Check: `Privacy.tsx` rendert in DE/EN/FR fehlerfrei (Sprache via Header-LanguageSwitcher togglen)
- Inhaltsabgleich: jede der 24 Tätigkeiten aus `PROCESSING-RECORDS.md` Sektion 2 ist im rendered HTML auffindbar
- Drittanbieter-Tabelle: 17 Einträge sichtbar (entspricht Sektion 4)
- Beschwerderecht-Sektion enthält Hinweis auf zuständige deutsche Aufsichtsbehörde (Bundesland-abhängig — typisch BfDI für bundesweit agierende Online-Dienste oder Landesbeauftragter am Sitz des Verantwortlichen)
- E2E-Smoke: Playwright-Test "privacy page renders all sections in DE" (optional)

## Out of scope

- Cookie-Policy (separater Plan 41-04)
- Imprint-Pflicht TMG §5 (eigene Seite `/imprint` — nicht Teil von Privacy.tsx; ggf. separater Folge-Plan oder durch 41-01 ergänzt)
- Anwalt-Review der Übersetzungs-Genauigkeit (extern, vor Produktion empfehlenswert)
