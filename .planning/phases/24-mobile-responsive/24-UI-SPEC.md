## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, pill gaps |
| md | 16px | Default element spacing, mobile content padding (p-4) |
| lg | 24px | Desktop content padding (p-6), section gaps |
| xl | 32px | Layout gaps between major sections |
| 2xl | 48px | Page-level spacing |
| 3xl | 64px | Reserved for large breakpoints |

- **44px**: Minimum touch target size (D-63)

- **72px**: FAB bottom offset (56px nav + 16px gap) (D-31)

## Typography

| Role | Size | Weight | Line Height | Font |
|------|------|--------|-------------|------|
| Body | 14px | 400 (regular) | 1.5 | Inter |
| Label | 10-12px | 400 (regular) | 1.4 | JetBrains Mono |
| Heading | 18px | 600 (semibold) | 1.3 | JetBrains Mono |
| Display | 24px | 600 (semibold) | 1.2 | JetBrains Mono |

**Weight rationale:** 2 weights only (regular + semibold) to prevent visual noise. Labels use regular weight with size differentiation. Display text uses semibold with size differentiation from headings.

**Mobile-specific adjustments:**
- Card titles: 18px (vs 20px on desktop) for single-column layout
- Navigation labels: 10px with 0.1em letter-spacing (D-04)
- Stat counters: Use existing signal-counter class

## Mobile Navigation Contract

### Bottom Navigation (D-01 through D-15)

```
+--------------------------------------------------+
|  Dashboard  |  Monitor  |  Events  |  Bookmarks  |  Profile  |
|     [*]          [ ]        [3]         [ ]          [ ]      |
+--------------------------------------------------+
```

**Visual specifications:**
- Height: 56px + safe-area-inset-bottom (env(safe-area-inset-bottom))
- Background: glass effect (rgba(10,14,26,0.8) + backdrop-blur(16px))
- Top border: 1px solid rgba(0,240,255,0.2)
- Icons: Lucide (Activity, Globe2, MapPin, Bookmark, User)
- Active state: filled icon + label + cyan glow
- Inactive state: outline icon only, no label
- Badge on Events: purple pill with event count

**Behavior:**
- Auto-hide on scroll down (threshold: 50px)
- Reappear on scroll up (threshold: 20px)
- Double-tap Dashboard scrolls to top
- Light haptic on tap (navigator.vibrate(10))
- Spring animation on active icon (Framer Motion)

### Mobile Header (D-16 through D-22)

```
+--------------------------------------------------+
|  [=]  |  [Radio] NEWSHUB  |  [Search]  |  [Avatar]  |
+--------------------------------------------------+
```

**Visual specifications:**
- Height: 56px (h-14), fixed position
- Background: header-cyber class (existing)
- Hamburger: Menu icon, left side
- Logo: Radio icon, taps to Dashboard
- Search: collapses to icon, expands full-width on tap
- Avatar: shows user initial or User icon

**Hidden on mobile:**
- FocusSelector (moved to drawer)
- FeedManager (moved to drawer)
- LanguageSwitcher (moved to drawer)
- Clock/connection status
- Keyboard shortcut hints

### Mobile Drawer (D-23 through D-27)

**Visual specifications:**
- Width: 280px (w-64 + extra for user info)
- Slide-in from left with spring animation
- Backdrop: black/60 with blur
- Close: swipe right, back button, X button, backdrop tap

**Contents (top to bottom):**
1. User info section (avatar + name if authenticated)
2. All navigation items (full duplication of bottom nav + extras)
3. FocusSelector
4. FeedManager
5. LanguageSwitcher
6. Settings link
7. Version + legal links

**NOT included:** Event stats panel (navigation focus only)

## Touch Interactions Contract

### Gestures (D-60 through D-72)

| Gesture | Target | Action | Feedback |
|---------|--------|--------|----------|
| Pull down | News feed | Refresh | Spinning Radio icon with cyan glow |
| Swipe right | News card | Bookmark | Spring reveal of cyan bookmark icon |
| Edge swipe (0-20px) | Left edge | Open drawer | Drawer slides in |
| Swipe right | Drawer content | Close drawer | Drawer slides out |
| Swipe down | Bottom sheets | Dismiss | Sheet slides down |
| Pinch | Article images | Zoom | Full-screen lightbox |
| Pan | Maps | Move view | Standard map behavior |
| Pinch | Maps | Zoom | Standard map behavior |
| Double-tap | Dashboard icon | Scroll to top | Smooth scroll |

### Touch Targets

- Minimum size: 44x44px (D-63)
- Touch feedback: subtle highlight (active:bg-white/5)
- CSS properties:
  - `touch-action: manipulation` (eliminate 300ms delay) (D-71)
  - `overscroll-behavior: contain` (prevent iOS rubber-band) (D-72)
  - `-webkit-overflow-scrolling: touch` (native scroll) (D-66)

## Image Optimization Contract (D-73 through D-82)

### ResponsiveImage Component Spec

```tsx
interface ResponsiveImageProps {
  src: string;
  alt: string;
  sizes: string;           // e.g., "(max-width: 768px) 100vw, 50vw"
  priority?: boolean;      // First 3 cards = true
  className?: string;
  aspectRatio?: '16:9' | '4:3' | '1:1';
}
```

**srcSet breakpoints:** 320w, 640w, 960w, 1280w (D-73)

**Format priority:** AVIF > WebP > JPEG (D-75)

**Loading strategy:**
- First 3 cards: `loading="eager"`, `fetchpriority="high"` (D-77, D-78)
- Rest: `loading="lazy"`, `decoding="async"` (D-79, D-80)

**Placeholder:** blur-up with 300ms fade (D-76)
- Tiny 10px wide image scaled up with CSS blur
- OR blurhash string (Claude's discretion)

**Transformation service:** Cloudinary (D-74)
- URL pattern: `https://res.cloudinary.com/{cloud}/image/fetch/f_auto,q_auto,w_{width}/{original_url}`

**Caching:** Use existing PWA cache strategy (7-day TTL) (D-82)

## Accessibility Contract

### Touch Accessibility

- All interactive elements: min 44x44px (D-63)
- Focus indicators: cyan outline (existing)
- Reduced motion: respect `prefers-reduced-motion`
  - Disable: spring animations, parallax, continuous animations
  - Keep: instant state changes, opacity transitions

### Screen Reader

- Bottom nav: `role="navigation"`, `aria-label="Main navigation"`
- Active nav item: `aria-current="page"`
- FAB: `aria-label="Scroll to top"`
- Drawer: `role="dialog"`, `aria-modal="true"`, `aria-label="Navigation menu"`
