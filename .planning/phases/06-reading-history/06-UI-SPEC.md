## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, badge padding |
| md | 16px | Default element spacing, card padding |
| lg | 24px | Section padding, card gaps |
| xl | 32px | Layout gaps between major sections |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing (not used in this phase) |

Exceptions:
- Touch targets for avatar picker: 44px minimum (accessibility)
- Podium heights: 72px, 100px, 140px (rank 3, 2, 1 respectively — all multiples of 4)

## Interaction Patterns

### History Page Filters

| Filter | Type | Options | URL State |
|--------|------|---------|-----------|
| Search | Text input | Debounce 300ms | `?q=` |
| Region | Multi-select dropdown | All 6 regions | `?regions=` |
| Date Range | Preset + Custom picker | Today, Yesterday, 7d, 30d, Custom | `?from=&to=` |
| Sentiment | Single-select | All, Positive, Negative, Neutral | `?sentiment=` |

**Pagination:** Infinite scroll (matches existing NewsFeed pattern)
**Clear filters:** Reset to defaults, update URL

### For You Carousel

| Interaction | Behavior |
|-------------|----------|
| Horizontal scroll | CSS scroll-snap, momentum |
| Card click | Open article in new tab |
| Refresh | Auto-refresh with main feed (per D-20) |
| Exclude read | Already-read articles hidden (per D-21) |

### Avatar Picker

| Interaction | Behavior |
|-------------|----------|
| Hover | Show tooltip with name + era |
| Click unlocked | Select avatar, highlight border |
| Save | Requires API call, show loading state |
| Upload | File picker, validate 2MB + JPEG/PNG, auto-resize |

### Leaderboard

| Interaction | Behavior |
|-------------|----------|
| Tab switch | Filter by time period, maintain scroll |
| User's row | Always visible, pinned at bottom |
| Opt-out toggle | In Settings, removes from leaderboard |
