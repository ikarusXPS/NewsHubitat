      redis-server
      --appendonly yes
      --appendfsync everysec

      --save 300 10

volumes:

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Host -> Container network | Docker bridge network isolates services |
| .env file -> Container | Secrets injected at runtime via env_file |
| Container -> Database | Internal network, not exposed to host |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-17-08 | Information Disclosure | .env file | mitigate | .env not committed (in .gitignore), env_file for runtime injection |
| T-17-09 | Denial of Service | depends_on | mitigate | service_healthy condition prevents premature startup |
| T-17-10 | Spoofing | Container names | accept | Internal Docker network, names only resolve within compose |
