<!-- ICU MessageFormat syntax for pluralization (per D-09) -->

Basic plural:
```
"{count, plural, =0 {No items} one {# item} other {# items}}"
```

German plural (note: German uses different grammar):
```
"{count, plural, =0 {Keine Artikel} one {# Artikel} other {# Artikel}}"
```

Interpolation:
```
"Hello, {name}!"
"Hallo, {name}!"
```

Select (gender, etc.):
```
"{gender, select, male {He} female {She} other {They}} liked this"
```
