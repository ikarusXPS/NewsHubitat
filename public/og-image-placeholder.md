# OG Image Placeholder

A proper og-image.png (1200x630px) should be created with:
- Dark background (#0a0a0f or #050810)
- NewsHub logo/branding
- Cyan accent color (#00f0ff)
- Text: "Multi-Perspective News Analysis"

For now, copy an existing image or create via design tools.
Recommended dimensions: 1200x630px for optimal Twitter/Facebook display.

## Technical Requirements

- Format: PNG (preferred) or JPEG
- Size: 1200x630 pixels (1.91:1 aspect ratio)
- File size: Under 5MB (ideally under 1MB)
- Location: /public/og-image.png

## Usage

This image is used as:
1. Default fallback when articles don't have thumbnails
2. Homepage social preview when sharing the main site URL
3. Referenced in index.html and sharingService.getOpenGraphTags()
